### Book Class Details

**Attributes:**
- title : str
- author : str
- isbn : str
- price : float
- publisher : str
- genre : str
- page_count : int
- publication_year : int

**Methods:**
- `display_info()` : Prints all information about the book.
- `read_sample()` : Simulates reading a short excerpt from the book.
- `get_book_age()` : Calculates the book's age based on the current year.

> **Note:**
